# Marth Systems Logo and Favicon Assets

Generated package for `marth.systems`.

## Folders

- `svg/` - vector logo assets and favicon SVG.
- `png/` - rendered PNG logo assets.
- `favicon/` - favicon.ico, PNG icon sizes, Apple touch icon, Android icons, and webmanifest.
- `original-crops/` - separated raster crops from the originally generated combined logo image.

## Recommended website usage

Use the SVG logo where possible:

```html
<img src="/brand/logo-horizontal-transparent.svg" alt="Marth Systems" width="300" height="80">
```

Use the favicon files by copying the contents of `favicon/` into your public/static root and adding `favicon/html-head-snippet.html` to your document head.

## Note

The `original-crops/` files are separated from the combined generated image. The `svg/`, `png/`, and `favicon/` files are clean production starter assets based on the same logo direction.
